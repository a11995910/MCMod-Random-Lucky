# Random Lucky Mod

## 📖 项目简介

Random Lucky Mod 是一个为 Minecraft 1.20.1 设计的 Forge 模组，旨在为游戏增加更多有趣的随机元素和增强功能。该模组提供了多种独特的物品和合成配方，让玩家的游戏体验更加丰富多彩。

## ✨ 核心功能

### 🎲 随机物品合成
- **合成配方**: 9个木棍排列成3x3网格
- **功能**: 合成后获得随机物品，增加游戏的惊喜元素
- **特点**: 每次合成都可能获得不同的物品，从基础材料到稀有装备

### ⚔️ 装备强化系统
- **合成配方**: 三个相同的装备品（横向或纵向排列）
- **功能**: 强化中间位置的装备，提升其属性
- **支持装备**: 剑、镐、斧、锹、锄头、头盔、胸甲、护腿、靴子
- **强化效果**: 增加耐久度、攻击力、防御力等属性

### ❤️ 血量增强面包
- **合成配方**: 钻石 + 面包
- **功能**: 食用后永久增加1颗心（2点血量）
- **上限**: 最多可增加400颗心
- **特点**: 提供持久的生存优势

### 🍎 超级苹果
- **获取方式**: 通过随机合成或特定配方获得
- **功能**: 提供强大的永久效果
- **特点**: 稀有且强力的增益物品

### 🍽️ 大胃王 (v1.1.0新增)
- **合成配方**: 腐肉 + 小麦种子 + 木棍（横向排列）
- **功能**: 食用后永久增加1格饱食度上限
- **特点**: 
  - 可以多次食用，没有上限限制
  - 使用NBT数据存储，确保数据持久化
  - 食用后显示增加信息和当前总额外上限
  - 恢复4点饱食度，饱和度修饰符0.6f

## 📦 安装说明

### 前置要求
1. Minecraft 1.20.1
2. Minecraft Forge 47.3.0 或更高版本
3. Java 17 或更高版本

### 安装步骤
1. 下载最新版本的 Random Lucky Mod
2. 将 `.jar` 文件放入 Minecraft 的 `mods` 文件夹
3. 启动游戏并享受新功能

## ⚖️ 游戏平衡性

### 设计理念
- **风险与收益**: 随机合成提供不确定性，增加游戏刺激性
- **资源消耗**: 所有增强功能都需要消耗相应资源
- **渐进式增强**: 血量和饱食度增强需要持续投入
- **策略选择**: 玩家需要权衡不同增强路径的优先级

### 平衡机制
- **材料成本**: 强化功能需要消耗稀有材料
- **时间投入**: 获得显著增强需要长期积累
- **选择权衡**: 不同功能之间存在资源竞争
